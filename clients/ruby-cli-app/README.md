# Time Travel Ruby CLI Demo

## Install
   ```bash
   bundle install
   ```

## How to run?

Before launching CLI app, start Time Travel in your docker-compose

```bash
MULTIPLAYER_OTLP_KEY="your-key" ENVIRONMENT="staging" bundle exec ruby main.rb
```
